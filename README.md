# Great British Baking Show Theme

This is a custom Omeka S theme inspired by "The Great British Baking Show," designed to create a warm, friendly, and appealing layout for Omeka S.

## Features
- Customizable layouts
- Pastel color schemes
- Easy-to-use folder structure

## Installation
1. Download or clone this repository.
2. Zip the folder and upload it via the Omeka S admin panel under `Appearance > Themes > Upload`.

Enjoy creating a collection as inviting as your favorite baked goods!
